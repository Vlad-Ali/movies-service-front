# Документация для запуска movies-service-front локально

## Предварительные требования

- Node.js (версия 14 или выше)
- npm (обычно устанавливается вместе с Node.js)

## Установка зависимостей

Перед первым запуском установите все необходимые зависимости:

```bash
npm install
```

## Настройка переменных окружения

1. В корне проекта создайте файл `.env`
2. Добавьте в него переменную для указания адреса API для backend:

```env
REACT_APP_API_URL=http://localhost:8080
```

**Примечание:** Замените `http://localhost:8080` на адрес вашего бэкенд-сервера, если он работает на другом порту.

## Доступные скрипты

### `npm run build`

Собирает проект для production в папку `build`:

```bash
npm run build
```

### `npm start`

Запускает приложение в режиме разработки:

```bash
npm start
```

После выполнения этой команды приложение будет доступно по адресу: [http://localhost:3000](http://localhost:3000)

## Полный процесс запуска

1. Установите зависимости (если ещё не сделали):
   ```bash
   npm install
   ```

2. Создайте и настройте файл `.env`

3. Запустите приложение:
   ```bash
   npm start
   ```

4. Для production-сборки:
   ```bash
   npm run build
   ```

## Примечания

- Файл `.env` должен находиться в корневой директории проекта
- После изменения файла `.env` перезапустите сервер разработки
- При использовании `npm run build` создается оптимизированная сборка в папке `build/`
- Для запуска production-сборки локально можно использовать серверы вроде `serve`
